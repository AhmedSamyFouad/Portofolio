About Me

Software Quality Advocate

Ahmed Samy is a dedicated and detail-oriented Software Tester with a genuine passion for ensuring software quality and delivering exceptional user experiences. He combines sharp analytical thinking with a meticulous eye for detail, enabling him to uncover even the most subtle potential issues and rigorously validate software functionality against business requirements.   • 3
    Enter Valid PasswordType correct password. Confirm input is masked (dots/asterisks) for security.

    Verify Dashboard LoadConfirm user is redirected to dashboard with correct welcome message and role-based UI.

Product Search Functionality
Product Search High Priority Functional ✓ Pass

Verify that users can search for products using keywords and that the system returns accurate, relevant results from the product catalog.

  • 1
    Navigate to HomepageOpen browser and load homepage. Confirm search bar is prominently visible and interactive.
  • 2

  • 3
    Trigger Search ActionPress Enter or click search icon. Observe page behavior, loading indicators, and URL changes.
